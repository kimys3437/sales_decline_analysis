
# Sales Decline Analysis

## Project Goal
Analyze the causes of sales decline using transaction data.

## Dataset
Online Retail Dataset (UCI Machine Learning Repository)

The dataset contains transaction records of a UK-based online retailer.

## Tools
- SQL (SQLite)
- Python (pandas, matplotlib)

## Analysis Process
1. Data cleaning
2. Monthly sales aggregation
3. Sales decomposition

Sales = Order Count ¡¿ Average Order Value

4. Country sales analysis
5. Visualization

## Key Findings

1. Sales growth was mainly driven by order count increase.
2. Average order value showed relatively small variation.
3. Sales peaked in November 2011 due to seasonal demand.
4. The December decline is caused by incomplete data.
